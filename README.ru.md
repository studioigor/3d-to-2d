# 3D в 2D конвертер

> **[Студия Игор](https://www.youtube.com/@studioigor)**

[English version](README.md)

Веб-приложение для конвертации 3D-моделей в 2D спрайт-листы. Загрузите модель, выберите анимации и ракурсы — получите готовый спрайт-лист для вашей игры.

## Возможности

- Загрузка 3D-моделей (GLB, GLTF, FBX) через drag-and-drop или выбор файла
- 3D-превью с вращением камеры и пресетами ракурсов (спереди, сзади, слева, справа и т.д.)
- Воспроизведение анимаций с контролем скорости
- Два режима камеры: перспектива и ортографическая
- Настройка освещения: свой источник света с регулировкой интенсивности и позиции
- Пакетный экспорт: выбор нескольких анимаций и нескольких ракурсов одновременно
- Настройка разрешения кадра (128 / 256 / 512 / 1024 px) и количества кадров
- Квадратная рамка в превью — видно точные границы рендера
- Прогресс-бар генерации
- Предпросмотр результата
- Скачивание в PNG или ZIP (спрайт-лист + отдельные кадры)
- Интерфейс на русском и английском с автоопределением языка

## Запуск

Приложение доступно онлайн: **[studioigor.github.io/3d-to-2d](https://studioigor.github.io/3d-to-2d/)** — установка не требуется.

Для локального запуска — никакой сборки не нужно, просто откройте через любой локальный сервер:

```bash
# Node.js
npx serve .

# Python
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000` (или `http://localhost:3000` для serve).

## Стек

- [Three.js](https://threejs.org/) — 3D-рендеринг
- [JSZip](https://stuk.github.io/jszip/) — создание ZIP-архивов
- [FileSaver.js](https://github.com/nicolo-ribaudo/FileSaver.js) — скачивание файлов
- Vanilla JS, CSS, HTML — без фреймворков и сборщиков

## Структура

```
├── index.html              — главная страница
├── css/style.css           — стили (тёмная тема)
└── js/
    ├── app.js              — логика UI, обработка событий
    └── modules/
        ├── i18n.js         — локализация RU/EN
        ├── scene.js        — Three.js сцена, камеры, освещение
        ├── loader.js       — загрузка и нормализация моделей
        └── capture.js      — генерация спрайт-листов, экспорт ZIP
```
